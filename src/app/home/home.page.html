So funktioniert es

Sie erstellen einfach ein Projekt, das Sie für einen Kunden umsetzen. Danach senden Sie Ihrem Kunden einen Link. Dieser verschafft ihm ohne weitere Registrierung Zugriff auf das Projekt. Nun können Sie To-Dos für das Projekt erstellen und verwalten. Sie können den Status der To-Dos ändern, sodass Ihr Kunde jederzeit sieht, wann das Projekt fertiggestellt wird und wie weit es fortgeschritten ist. Zudem können Ihre Kunden direkt Feedback geben oder selbst To-Dos hinzufügen.

Starten Sie jetzt – kostenlos

Erstellen Sie einfach ein Konto und Ihr kostenloser Test (maximal 1 Kunde) beginnt sofort. Nun können Sie einen Kunden hinzufügen und Projekte erstellen. Senden Sie Ihrem Kunden den Einladungslink, und die einfache Kommunikation kann beginnen!
